# Real Estate System

### Σύστημα διαχείρισης ακινήτων υλοποιημένο σε Spring Boot

## Τεχνολογίες που χρησιμοποιούνται:

- **Thymeleaf**
- **Spring Security**
- **Spring Data JPA**
- **Hibernate**

## Απαιτούμενα:

- **Java 17**
- **Maven**
- **MySQL server**
- **Git**
- **Συμβατό IDE (π.χ. IntelliJ IDEA, Eclipse, VS Code με Java extensions)**
